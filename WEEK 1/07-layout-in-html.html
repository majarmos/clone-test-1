Now that we have learned all of this amazing elements, how can we distribute them?

Wait, before we learn how to do this guys, we need to learn a little bit of what html semantic & non-semantic elements are!

Semantic Elements = elements that their name provide context of their usage to both the programmer and the browser

Non-Semantic Elements = elements whose names do not provide any context, and they can be used to mark up semantics or to basically group other elements together in your own.

So far we have learned how to use html elements to place content within the browser, you see all of these elements are semantic in nature because their name describe their meaning

Although we can apply some semantics through applying an identifier or a class to any given element, we can use semantic elements to distribute content throghout our file according to what we are trying to achieve through our user.

I.E: We need to provide a way for our users to navigate through our website, so we have to create a navigation section for our users. Luckily HTML, provides us with some elements that help us provide a good layout to our site.
